# Выполненные задачи — 28.02.2026

## API Endpoints (новые)

1. POST /auth/2fa/setup — настройка двухфакторной аутентификации (TOTP)
2. POST /auth/2fa/verify — подтверждение 2FA кодом из authenticator
3. POST /auth/2fa/disable — отключение 2FA (с подтверждением пароля)
4. POST /tasks/batch — пакетная загрузка до 100 задач за раз
5. GET /tasks/calendar — задачи сгруппированные по дням
6. GET /tasks/stats/summary — агрегация задач по статусам
7. GET /tasks/{id}/progress — Redis-backed прогресс загрузки (stage + %)
8. GET /tasks/{id}/preview — инфо о файле + YouTube URL
9. PUT /tasks/{id} — перепланирование задачи (смена даты/времени)
10. POST /tasks/{id}/cancel — отмена запланированной публикации
11. GET /tasks/history — история завершённых/ошибочных/отменённых задач
12. CRUD /templates/ — шаблоны расписаний с time slots по дням недели
13. POST /templates/{id}/slots — добавление слота в шаблон
14. DELETE /templates/{id}/slots/{slot_id} — удаление слота
15. GET /channels/{id}/stats — ежедневная статистика канала
16. GET /admin/dashboard — обзор системы (каунты, очереди)
17. GET /admin/users — все пользователи с статистикой
18. GET /admin/queue — статус Redis очередей
19. POST /uploads/video — загрузка видео (mp4, mkv, avi и т.д., до 10GB)
20. POST /uploads/thumbnail — загрузка превью (jpg, png, webp, до 50MB)

## Security

21. Rate limiting — 10 req/min логин, 5 req/min регистрация, 120/min default
22. Security headers — X-Frame-Options, X-Content-Type-Options, X-XSS-Protection, Referrer-Policy
23. CORS ограничены — только нужные методы и заголовки
24. Path traversal protection — блокировка `..` в file paths
25. Error sanitization — никаких stack traces в ответах
26. Swagger /docs скрыт при DEBUG=False (потом включили обратно)
27. Audit logging — все auth events + task operations → /var/log/cff-audit.log
28. 2FA backup codes — 8 одноразовых кодов при включении TOTP
